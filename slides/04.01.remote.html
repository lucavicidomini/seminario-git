Git clone

  • Posso clonare un repository con il comando clone
  • Il clone conterrà una copia di indice, storia e del repository originale
    • La working directory sarà un checkout di HEAD
  • Il clone avrà un riferimento al repository originale che chiamerà origin

Branch remoti

  • Uno dei principi di git è la collaborazione
  • Più utenti possano lavorare su un repository condiviso – ad esempio su GitHub
  • Quindi, di default, i branch remoti che non sono sul repository locale vengono nascosti
